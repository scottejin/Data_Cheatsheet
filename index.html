1. Where to start

If you are new, here is the easiest mental model:

  • pandas is for loading, cleaning, filtering, grouping, and preparing data.
  • matplotlib is the underlying plotting library. It is dependable and flexible.
  • seaborn sits on top of matplotlib and makes many statistical plots easier and prettier.
Best beginner rule: if you are in an exam and feel stuck, start with pandas for cleaning, then use either df.plot(...) or seaborn for the graph.

Basic setup code

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

sns.set_theme()  # optional, but makes plots look nicer

Typical data load

df = pd.read_csv("my_data.csv")
print(df.head())
print(df.info())

2. A simple exam workflow

  1. Load the data with pd.read_csv() or pd.read_excel().
  2. Inspect it using head(), info(), describe(), and isna().sum().
  3. Clean it by fixing missing values, duplicates, wrong data types, and messy text.
  4. Pick the right graph based on the question being asked.
  5. Label everything clearly: title, axes, legend.
  6. Explain what the graph shows in plain language if the exam asks for interpretation.
Good habit: never jump straight into graphing. A graph based on dirty data can be misleading even if the code runs perfectly.

3. Clean data before graphing

Many graphing problems are actually data cleaning problems in disguise. Before you plot, check these things.

3.1 Missing values

df.isna().sum()

Use this to see how many missing values each column has.

# remove rows with missing values in important columns
clean_df = df.dropna(subset=["sales", "date"])

# fill missing values
clean_df["region"] = clean_df["region"].fillna("Unknown")

3.2 Duplicates

df.duplicated().sum()
df = df.drop_duplicates()

3.3 Wrong data types

df.info()

df["date"] = pd.to_datetime(df["date"])
df["sales"] = pd.to_numeric(df["sales"], errors="coerce")

3.4 Messy text

df["city"] = df["city"].str.strip().str.title()
df["category"] = df["category"].str.lower()

3.5 Outliers

Outliers are unusually large or small values. They are not always wrong, but they can strongly affect your graph. Box plots are especially useful for spotting them.

Important: do not delete outliers automatically unless you have a valid reason. In an exam, it is better to explain why you kept or removed them.

5. Line charts

Use a line chart when: your x-axis has a meaningful order, especially time.

What it shows

A line chart helps you see trends, increases, decreases, and patterns over time.

Simple example with pandas

df = pd.DataFrame({
    "month": ["Jan", "Feb", "Mar", "Apr"],
    "sales": [120, 150, 140, 180]
})

plt.plot(df["month"], df["sales"])
plt.title("Monthly Sales")
plt.xlabel("Month")
plt.ylabel("Sales")
plt.show()

Better matplotlib version

fig, ax = plt.subplots()
ax.plot(df["month"], df["sales"], marker="o")
ax.set_title("Monthly Sales")
ax.set_xlabel("Month")
ax.set_ylabel("Sales")
ax.grid(True)
plt.show()

Beginner explanation

  • marker="o" puts a dot on each data point.
  • set_title() adds the graph title.
  • set_xlabel() and set_ylabel() label the axes.
  • grid(True) makes it easier to read values.
Tip: line charts are usually not good for unrelated categories like "Apple", "Banana", and "Orange". Use a bar chart instead.

6. Bar charts

Use a bar chart when: you want to compare amounts across categories.

What it shows

Bar charts make it easy to compare sizes between groups, such as sales by product or number of students by class.

Basic bar chart

df = pd.DataFrame({
    "product": ["A", "B", "C"],
    "sales": [40, 55, 32]
})

plt.bar(df["product"], df["sales"])
plt.title("Sales by Product")
plt.xlabel("Product")
plt.ylabel("Sales")
plt.show()

Horizontal bar chart

plt.barh(df["product"], df["sales"])
plt.title("Sales by Product")
plt.xlabel("Sales")
plt.ylabel("Product")
plt.show()

When horizontal bars are better

If category names are long, a horizontal bar chart is usually easier to read.

Common mistake: do not use a bar chart for raw continuous values over time if a line chart would show the trend more clearly.

7. Histograms

Use a histogram when: you want to see the distribution of one numeric variable.

What it shows

A histogram groups numeric values into bins and shows how many values fall into each bin.

Example

scores = [45, 50, 52, 60, 61, 63, 70, 72, 74, 80, 83, 90]

plt.hist(scores, bins=5, edgecolor="black")
plt.title("Distribution of Scores")
plt.xlabel("Score")
plt.ylabel("Frequency")
plt.show()

Why bins matters

The bins argument controls how many groups the data is split into. Too few bins can hide detail. Too many bins can make the graph noisy.

What to look for

  • Is the data roughly symmetrical?
  • Is it skewed left or right?
  • Are there multiple peaks?
  • Are there gaps or extreme values?

8. Scatter plots

Use a scatter plot when: you want to examine the relationship between two numeric variables.

What it shows

Each point represents one row of data. You can look for positive relationships, negative relationships, clusters, or outliers.

Matplotlib example

df = pd.DataFrame({
    "study_hours": [1, 2, 3, 4, 5, 6],
    "score": [50, 55, 65, 70, 78, 88]
})

plt.scatter(df["study_hours"], df["score"])
plt.title("Study Hours vs Exam Score")
plt.xlabel("Study Hours")
plt.ylabel("Score")
plt.show()

Seaborn version

sns.scatterplot(data=df, x="study_hours", y="score")
plt.title("Study Hours vs Exam Score")
plt.show()

What to say in an exam

If the dots rise from left to right, there may be a positive relationship. If they fall from left to right, there may be a negative relationship. If they are scattered randomly, there may be little or no relationship.

9. Box plots

Use a box plot when: you want to compare distributions across categories or identify outliers.

What it shows

  • Middle line: median
  • Box: middle 50% of the data
  • Whiskers: spread beyond the box
  • Dots beyond whiskers: possible outliers

Example

df = pd.DataFrame({
    "class": ["A", "A", "A", "B", "B", "B"],
    "score": [55, 60, 90, 62, 64, 66]
})

sns.boxplot(data=df, x="class", y="score")
plt.title("Score Distribution by Class")
plt.show()
Good use: compare salary distributions across departments, exam scores across classes, or delivery times across regions.

10. Violin plots

Use a violin plot when: you want to see the shape of the distribution, not just the quartiles.

Example

sns.violinplot(data=df, x="class", y="score")
plt.title("Score Distribution by Class")
plt.show()

How it differs from a box plot

A box plot gives a compact summary. A violin plot shows the density shape, so you can better see where values are concentrated.

Beginner note: if your teacher has not covered violin plots, do not force them into an exam answer. A box plot is often safer and easier to explain.

11. Count plots

Use a count plot when: you want the number of rows in each category.

Example

df = pd.DataFrame({
    "subject": ["Maths", "English", "Maths", "Science", "English", "Maths"]
})

sns.countplot(data=df, x="subject")
plt.title("Number of Students by Subject")
plt.show()

Why not just a bar chart?

You can use a bar chart too, but a count plot automatically counts category frequencies from the raw data.

12. Heatmaps

Use a heatmap when: you want to show values in a colour grid, especially correlations.

Correlation heatmap example

corr = df.corr(numeric_only=True)

sns.heatmap(corr, annot=True, cmap="coolwarm")
plt.title("Correlation Heatmap")
plt.show()

What this means

  • corr() calculates correlations between numeric columns.
  • annot=True prints the numbers inside the boxes.
  • cmap="coolwarm" sets the colour scheme.
Exam wording: a correlation heatmap helps identify variables that move together strongly, either positively or negatively.

13. Labels, legends, subplots, and saving your graph

Always label your graph

plt.title("Sales by Month")
plt.xlabel("Month")
plt.ylabel("Sales")

Add a legend when needed

ax.plot(x, y1, label="2024")
ax.plot(x, y2, label="2025")
ax.legend()

Use subplots for multiple graphs

fig, axes = plt.subplots(1, 2, figsize=(10, 4))

axes[0].hist(df["score"], bins=5)
axes[0].set_title("Scores")

axes[1].boxplot(df["score"])
axes[1].set_title("Score Spread")

plt.tight_layout()
plt.show()

Save your graph

plt.savefig("my_graph.png", dpi=300, bbox_inches="tight")

What this does:

  • dpi=300 makes the image higher quality.
  • bbox_inches="tight" trims unnecessary whitespace.

14. Common mistakes beginners make

Using the wrong graph
Example: using a pie chart or line chart when a bar chart would be clearer.
Not cleaning data first
Missing values, wrong types, and duplicate rows can distort the result.
No labels
A graph without a title or axis labels is much harder to understand.
Overcomplicating the graph
If a simple bar chart answers the question, do not make a fancy multi-layer plot.

Extra caution about pie charts

Pie charts exist, but they are often harder to compare accurately than bar charts. If your exam specifically asks for a pie chart, use one. Otherwise, a bar chart is often the better choice.

df["sales"].plot(kind="pie", autopct="%1.1f%%")
plt.ylabel("")
plt.show()

15. Practice tasks

You have monthly website visitors. Which graph should you use?

Answer: a line chart.

Answer: a bar chart.

16. Quick cheat sheet

# Load data
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
sns.set_theme()

df = pd.read_csv("data.csv")

# Inspect data
df.head()
df.info()
df.describe()
df.isna().sum()

# Clean data
df = df.drop_duplicates()
df["date"] = pd.to_datetime(df["date"], errors="coerce")
df["value"] = pd.to_numeric(df["value"], errors="coerce")

# Line chart
sns.lineplot(data=df, x="date", y="value")

# Bar chart
sns.barplot(data=df, x="category", y="value")

# Histogram
sns.histplot(data=df, x="value", bins=20)

# Scatter plot
sns.scatterplot(data=df, x="x_col", y="y_col")

# Box plot
sns.boxplot(data=df, x="group", y="value")

# Violin plot
sns.violinplot(data=df, x="group", y="value")

# Count plot
sns.countplot(data=df, x="category")

# Heatmap
corr = df.corr(numeric_only=True)
sns.heatmap(corr, annot=True, cmap="coolwarm")

plt.show()

17. Fully annotated code (line-by-line)

This section rewrites every core example with detailed comments. Read the comments directly in the code blocks — each line explains exactly what it does and why it is there.

A) Setup and imports

# Import pandas for data tables (DataFrame objects)
import pandas as pd

# Import numpy for numerical helpers and arrays
import numpy as np

# Import matplotlib's plotting interface
import matplotlib.pyplot as plt

# Import seaborn for higher-level statistical charts
import seaborn as sns

# Apply seaborn's default visual theme so charts look cleaner
sns.set_theme()

B) Load and inspect a dataset

# Read a CSV file from disk into a DataFrame named df
# Replace "my_data.csv" with your real file name/path
df = pd.read_csv("my_data.csv")

# Show first 5 rows so you can quickly inspect columns and sample values
print(df.head())

# Show column names, data types, and non-null counts
print(df.info())

# Show numeric summary (count, mean, std, min, quartiles, max)
print(df.describe())

C) Data cleaning examples (fully annotated)

# Count missing values in each column
missing_by_column = df.isna().sum()
print(missing_by_column)

# Drop rows where critical columns are missing
# Here, if 'sales' or 'date' is missing, that row is removed
clean_df = df.dropna(subset=["sales", "date"])

# Fill missing text values in 'region' with a default label
clean_df["region"] = clean_df["region"].fillna("Unknown")

# Count duplicate rows (True means a row repeats an earlier row)
duplicate_count = clean_df.duplicated().sum()
print("Duplicate rows:", duplicate_count)

# Remove duplicate rows
clean_df = clean_df.drop_duplicates()

# Convert 'date' text into proper datetime values
# errors='coerce' means invalid dates become NaT (missing datetime)
clean_df["date"] = pd.to_datetime(clean_df["date"], errors="coerce")

# Convert 'sales' to numeric
# errors='coerce' means invalid strings become NaN
clean_df["sales"] = pd.to_numeric(clean_df["sales"], errors="coerce")

# Standardize city text: strip spaces then title-case
clean_df["city"] = clean_df["city"].str.strip().str.title()

# Standardize category text: lower-case for consistency
clean_df["category"] = clean_df["category"].str.lower()

D) Line chart (matplotlib OO style)

# Create example data
line_df = pd.DataFrame({
    "month": ["Jan", "Feb", "Mar", "Apr"],
    "sales": [120, 150, 140, 180]
})

# Create a figure (the whole image) and one axes (the plotting area)
fig, ax = plt.subplots()

# Plot month on x-axis and sales on y-axis
# marker='o' draws circles on each data point
ax.plot(line_df["month"], line_df["sales"], marker="o")

# Add chart title
ax.set_title("Monthly Sales")

# Label x-axis
ax.set_xlabel("Month")

# Label y-axis
ax.set_ylabel("Sales")

# Show grid lines for easier reading
ax.grid(True)

# Display the chart window/output
plt.show()

E) Bar chart (vertical and horizontal)

# Example category data
bar_df = pd.DataFrame({
    "product": ["A", "B", "C"],
    "sales": [40, 55, 32]
})

# Vertical bar chart
plt.bar(bar_df["product"], bar_df["sales"])  # x=categories, y=values
plt.title("Sales by Product")                 # chart title
plt.xlabel("Product")                         # x-axis label
plt.ylabel("Sales")                           # y-axis label
plt.show()                                     # render chart

# Horizontal bar chart
plt.barh(bar_df["product"], bar_df["sales"]) # y=categories, x=values
plt.title("Sales by Product")
plt.xlabel("Sales")
plt.ylabel("Product")
plt.show()

F) Histogram

# One numeric list to visualize as a distribution
scores = [45, 50, 52, 60, 61, 63, 70, 72, 74, 80, 83, 90]

# bins=5 splits value range into 5 groups
# edgecolor draws outlines around bars for readability
plt.hist(scores, bins=5, edgecolor="black")

# Add readable labels
plt.title("Distribution of Scores")
plt.xlabel("Score")
plt.ylabel("Frequency")

# Show chart
plt.show()

G) Scatter plot (matplotlib and seaborn)

# Example numeric relationship data
scatter_df = pd.DataFrame({
    "study_hours": [1, 2, 3, 4, 5, 6],
    "score": [50, 55, 65, 70, 78, 88]
})

# Matplotlib scatter: one dot per row
plt.scatter(scatter_df["study_hours"], scatter_df["score"])
plt.title("Study Hours vs Exam Score")
plt.xlabel("Study Hours")
plt.ylabel("Score")
plt.show()

# Seaborn scatter (same idea, dataframe-friendly API)
sns.scatterplot(data=scatter_df, x="study_hours", y="score")
plt.title("Study Hours vs Exam Score")
plt.show()

H) Box plot and violin plot

# Category + numeric values for distribution comparison
dist_df = pd.DataFrame({
    "class": ["A", "A", "A", "B", "B", "B"],
    "score": [55, 60, 90, 62, 64, 66]
})

# Box plot: median, quartiles, whiskers, possible outliers
sns.boxplot(data=dist_df, x="class", y="score")
plt.title("Score Distribution by Class")
plt.show()

# Violin plot: distribution shape + density
sns.violinplot(data=dist_df, x="class", y="score")
plt.title("Score Distribution by Class")
plt.show()

I) Count plot

# Category-only data
count_df = pd.DataFrame({
    "subject": ["Maths", "English", "Maths", "Science", "English", "Maths"]
})

# countplot automatically counts rows per category
sns.countplot(data=count_df, x="subject")
plt.title("Number of Students by Subject")
plt.show()

J) Correlation heatmap

# Keep only numeric columns and compute correlation matrix
corr = clean_df.corr(numeric_only=True)

# Draw heatmap of correlation values
# annot=True prints numbers inside cells
# cmap selects color palette
sns.heatmap(corr, annot=True, cmap="coolwarm")
plt.title("Correlation Heatmap")
plt.show()

K) Labels, legends, subplots, and save

# Suppose x, y1, y2 are already defined arrays/lists
fig, ax = plt.subplots()              # make figure + axes
ax.plot(x, y1, label="2024")          # first line with legend label
ax.plot(x, y2, label="2025")          # second line with legend label
ax.set_title("Sales by Month")        # title
ax.set_xlabel("Month")                # x-axis label
ax.set_ylabel("Sales")                # y-axis label
ax.legend()                           # show legend from labels
plt.show()                            # draw

# Subplots: 1 row, 2 columns
fig, axes = plt.subplots(1, 2, figsize=(10, 4))
axes[0].hist(clean_df["score"], bins=5)   # left plot = histogram
axes[0].set_title("Scores")
axes[1].boxplot(clean_df["score"])        # right plot = boxplot
axes[1].set_title("Score Spread")
plt.tight_layout()                         # fix spacing overlap
plt.show()

# Save current figure as high-quality PNG
plt.savefig("my_graph.png", dpi=300, bbox_inches="tight")

L) Pie chart (only when specifically required)

# Create pie chart from a numeric series
# autopct formats percentages on slices
df["sales"].plot(kind="pie", autopct="%1.1f%%")

# Remove default y-label text for cleaner look
plt.ylabel("")

# Show pie chart
plt.show()
